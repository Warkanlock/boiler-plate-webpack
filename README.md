# Boiler Plate Webpack 
A boilerplate for starting a site without rush, tiny size and ideal for interviews development tasks (save time for that interview!)

## Preparation

``` npm install ```

## Build

``` npm run build ```

## Run and watch in real-time

``` npm run start ```
